Actualización de las imagenes de la página principal

Podremos tener hasta tres imágenes que irán deslizandose por la página inicial. Deberemos tener en cuenta que todas deberán tener las mismas demensiones y no ser demasiado pesadas pera no demorar la carga de las páginas. Para conseguirlo, podemos tratar las imágenes con algún software específico para ello.

Para actualizar las imágenes, deberemos registrarnos e iniciar sesión con perfil de administrador.
